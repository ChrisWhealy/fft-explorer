Where Do Sine and Cosine Waves Come From?

You may have been taught in school that "sine" or "cosine" are something to do with calculating the angles inside a triangle.

Or maybe you were just in the room when mathematics was being taught…

However, the heart of the matter is that this right-angled triangle you learnt about in school is simply a tool for understanding rotation and is constructed inside a circle where the triangle's longest side is the circle's radius.

To understand this, let's draw a circle with a radius of one unit whose centre (point `O`) is located at the origin of some coordinate system. Next, we mark a point `C` where the circumference of the circle crosses the positive X axis.

Now, as point `C` rotates in an anti-clockwise direction, we can see that a right-angled triangle is formed within the circle whose width is the distance between point `C` and some point `W` projected onto the Y axis, and whose height is the distance between point `C` and some point `H` projected onto the X axis.

As point `C` moves around the circle, point `W` moves up and down the Y axis, and point `H` moves left and right along the X axis. The key point to understand here is that the height (line `CH`) and width (line `CW`) change in a fixed ratio that is directly proportional to the angle of rotation.

The graphs to the right of and below the circle trace a wave shape that describes how the height and width of the triangle vary with the angle.

Also notice that a sine wave and a cosine wave have exactly the same shape; the only difference being that when the angle is zero, the sine wave has a value of zero, but the cosine wave has a value of one.

Trigonometric Functions

If we divide the height `CH` or the width `CW` of the triangle by the length of the radius `OC`, we obtain a fraction that exactly represents the angle of rotation. Also this relationship is a two-way street. If we know the angle, we can work out the ratio, and if we know the ratio, we can work out the angle.

The trigonometric functions are simply the names given to these ratios.

Sine

"Sine" is the name given to the ratio of the height `CH` of the triangle to the radius `OC` of the circle. This value varies in the range `-1..+1`.

There is an exact relationship between the angle θ and the ratio of height to radius. So when we use the function `sin(θ)`, we are really calculating height radius

Since we have arbitrarily defined our circle to have a radius of one, the sine of the angle is simply the height of the triangle.

Similarly, if we know the sine value of an angle, we can turn that fraction back into an angle using the inverse sine function: `θ = sin^-1 (value)`. (Also written `arcsin()`)

Cosine

"Cosine" is the name given to the ratio of the width `CW` of the triangle to the radius `OC` of the circle. This value varies in the range `-1..+1`.

There is an exact relationship between the angle θ and the ratio of width to radius. So when we use the function `cos(θ)`, we are really calculating width radius

Since we have arbitrarily defined our circle to have a radius of one, the cosine of the angle is simply the width of the triangle.

Similarly, if we know the cosine value of an angle, we can turn that fraction back into an angle using the inverse cosine function: `θ = cos^-1 (value)`. (Also written `arccos()`)

Tangent

Lastly, "tangent" is the name given to the ratio of the triangle's height `CH` to its width `CW`. The tangent value varies in the range `-∞..+∞`.

If we know the angle θ, then the value known as `tan(θ)` can be obtained by calculating height width

Similarly, if we know the tangent value of an angle, we can turn that value back into an angle using the inverse tangent function: `θ = tan^-1(value)` (Also written `arctan()`)
