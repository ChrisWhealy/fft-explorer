Convolution: Calculating A Component's Magnitude

As before, our signal is composed of three sine waves:

When the integral of the convolution wave reaches a maximum, we can use that value to calculate that component's magnitude.

Here, the test wave has been set to a frequency of 5Hz and a phase of 30°. This corresponds to the maximum of `90` for the 5Hz component within the signal

Component Magnitude Phase
5 Hz 0.5 30°
8 Hz 0.3 60°
10 Hz 0.5 80°

Signal + Test Wave

Convolution Curve

To calculate a component's magnitude, we use the following formula:

`Magnitude = 2 xx (max) / (360°)`

In this case, `max = 90`, so

`Magnitude = 2 xx 90 / (360°)`
`Magnitude = 0.5`

You can confirm this for yourself by setting the test wave phase and frequency to the values shown in the table above. First, confirm that the maximum occurs at the phase angle the table says it should, then plug the maximum value of the integral into the above equation.
