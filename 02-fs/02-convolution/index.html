Discovering Component Frequencies Within the Signal

Now let's look at what happens when the test wave does not match any frequency in the signal. Firstly, this means that the signal needs to contain multiple frequencies, and secondly, we need to be able to change the frequency of the test wave.

With the test wave set to 4Hz, you'll notice that no matter how you change the test wave's phase, the integral of the convolution curve stays at (or very close to) zero.

This is simply because there is no 4Hz component in the signal.

Now start changing the frequency of the test wave. Notice that as it approaches 5Hz, 8Hz or 10Hz, the integral of the convolution curve starts to grow.

Alternatively, as the frequency of the test wave diverges from 5Hz, 8Hz, or 10Hz, then the integral stays at (or very close to) zero.

What Does This Tell Us?

When we see a maximum in the integral of the convolution curve, this tells us three important things:

  1. The signal contains a component equal to the frequency of the test wave.
  2. This component in the signal occurs at a phase equal to the phase shift of the test wave.
  3. The maximum value of the integral is proportional to the amplitude of this component within the signal.
