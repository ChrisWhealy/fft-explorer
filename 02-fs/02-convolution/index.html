Convolution Part 1

Adjusting the Test Wave To Discover Component Frequencies Within the Signal

Now let's look at what happens when the test wave does not match any frequency in the signal. Firstly, this means that the signal needs to contain multiple frequencies, and secondly, we need to be able to change the frequency of the test wave.

The blue graph below shows a signal that is composed of three sine waves; 6Hz, 9Hz and 11Hz. To keep things simple, each of these frequencies is present at equal amplitudes.

With the test wave set to 5Hz, you'll notice that no matter how you change the test wave's phase, the integral of the convolution curve stays at (or very close to) zero.

Signal + Test Wave

Convolution Curve

Now start changing the frequency of the test wave. Notice that as it approaches 6Hz, 9Hz or 11Hz, the integral of the convolution curve starts to grow.

Alternatively, as the frequency of the test wave diverges from 6Hz, 9Hz, or 11Hz, then the integral stays at (or very close to) zero.

Integral of Convolution Curve

What Does This Tell Us?

When we see a maximum in the integral of the convolution curve, this tells us three important things:

  1. The signal contains a component equal to the frequency of the test wave.
  2. This component occurs at a phase equal to the phase shift of the test wave.
  3. The maximum value of the integral is proportional to the amplitude of this component within the signal.
