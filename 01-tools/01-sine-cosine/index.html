Where Do Sine and Cosine Waves Come From?

You may have been taught in school that "sine" or "cosine" are something to do with calculating the angles inside a triangle.

Or maybe you were just in the room when mathematics was being taught…

However, the heart of the matter is that this right-angled triangle you learnt about in school is simply a tool for understanding rotation and is constructed inside a circle where the triangle's longest side is the circle's radius.

To understand this, let's draw a circle with a radius of one unit whose centre (point `O`) is located at the origin of some coordinate system. Next, we mark a point `C` where the circumference of the circle crosses the positive X axis.

As the radius of the circle slowly rotates, the point `C` on the circumference forms the apex of a right-angled triangle with the X and Y axes where the height is the line `CH` and the width is the line `CW`.

As point `C` moves around the circle, the point `W` moves up and down the Y axis, and point `H` moves left and right along the X axis. The key point to understand here is that the height (line `CH`) and width (line `CW`) change in a fixed ratio that is exactly proportional to the angle of rotation.

The graphs to the right of and below the circle trace a wave shape that describes how the height and width of the triangle vary with the angle.

Also notice that a sine wave and a cosine wave have exactly the same shape; the only difference being that when the angle is zero, the sine wave has a value of zero, but the cosine wave has a value of one.

Trigonometric Functions

The trigonometric functions such as "sine", "cosine" and "tangent" are simply the names given to the ratios of the lengths of the sides inside this triangle.

In all of these examples, we have arbitrarily defined our circle to have a radius of `1`, so calculations involving the length of the hypotenuse disappear.

IMPORTANT
Always remember that first and foremost, these trigonometric functions describe rotation around a circle with a radius of `1`. The triangles we talk about here exist simply as a tool for calculating how the position of a point rotating around the circumference of the circle changes relative to the X and Y axes.

Sine

"Sine" is name given to the ratio of the triangle's height (`CH`) to its radius (`OC`).

This ratio varies from `-1` to `+1`.

So when we calculate height radius that ratio is known as the sine of the angle `θ` or `sin(θ)`.

Or, to use the more conventional names sin(θ) = opposite hypotenuse

Therefore, if we know the sine of an angle, we can turn that fraction back into an angle using the inverse sine function: `θ = sin^-1 (value)`. (Also written `arcsin()`)

Cosine

"Cosine" is name given to the ratio of the triangle's width (`CW`) to its radius (`OC`).

This ratio varies from `-1` to `+1`.

So when we calculate width radius that ratio is known as the cosine of the angle `θ` or `cos(θ)`.

Or, to use the more conventional names cos(θ) = adjacent hypotenuse

Therefore, if we know the cosine of an angle, we can turn that fraction back into an angle using the inverse cosine function: `θ = cos^-1 (value)`. (Also written `arccos()`)

Tangent

Lastly, "tangent" is the name given to the ratio of the triangle's height `CH` to its width `CW`.

The tangent value varies from `-∞` to `+∞`.

So when we calculate height width that ratio is known as the tangent of the angle `θ` or `tan(θ)`.

Or, to use the more conventional names cos(θ) = opposite adjacent

Therefore, if we know the tangent of an angle, we can turn that number back into an angle using the inverse tangent function: `θ = tan^-1 (value)`. (Also written `arctan()`)
